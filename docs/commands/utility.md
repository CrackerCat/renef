# Utility Commands

## `help`

Show available commands.

## `q`

Exit renef client.

## `clear`

Clear the terminal screen.

## `color <theme>=<COLOR>`

Set terminal color theme.

```bash
# List current theme
color

# Set prompt color
color prompt=CYAN

# Set response color
color response=GREEN
```
